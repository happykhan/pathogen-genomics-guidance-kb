## Not Added As Direct PDF Links

- PHA4GE infrastructure source pack: the public repository currently exposes Markdown source material rather than a single PDF download.
- PubMed records: PubMed provides bibliographic pages, not always direct publisher PDFs.
- PMC records: several candidate PDF paths returned HTML wrappers rather than `application/pdf`, so they were not exposed as direct PDF links.
- WHO publication pages: direct PDF downloads should be added after confirming stable WHO/IRIS file URLs for each publication.
