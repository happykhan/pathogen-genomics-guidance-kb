# Infrastructure Requirements Checklist

## Use Cases

- [ ] Routine surveillance workflows defined.
- [ ] Outbreak response workflows defined.
- [ ] Reporting requirements defined.
- [ ] Data sharing requirements defined.

## Compute

- [ ] Expected current throughput estimated.
- [ ] Expected future throughput estimated.
- [ ] Compute model selected: laptop/server/HPC/cloud/hybrid/service.
- [ ] Workflow execution system selected.
- [ ] Container/runtime approach defined.

## Storage

- [ ] Raw data volume estimated.
- [ ] Intermediate data volume estimated.
- [ ] Final output volume estimated.
- [ ] Retention periods defined.
- [ ] Backup requirements defined.
- [ ] Archive requirements defined.

## Operations

- [ ] Platform owner assigned.
- [ ] Workflow owner assigned.
- [ ] User support process defined.
- [ ] Monitoring/logging requirements defined.
- [ ] Update and validation process defined.
- [ ] Incident response contacts defined.

## Governance

- [ ] User roles defined.
- [ ] Authentication approach defined.
- [ ] Access review process defined.
- [ ] Offboarding process defined.
- [ ] Data export rules defined.
