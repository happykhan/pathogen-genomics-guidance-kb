# Implementation Vignettes

This folder is for short implementation case studies that illustrate different infrastructure models.

Initial candidate vignettes from the PHA4GE source pack:

- INRB laptop/field bioinformatics
- SANBI/UWC on-prem HPC
- University of Ibadan / Nextflow Tower
- Terra for US public health laboratories
- UseGalaxy.eu / GalaxyTrakr-style model
- NVI IRIDA

Each vignette should eventually use the same template:

- setting
- use case
- infrastructure model
- workflow model
- data flow
- access model
- strengths
- limitations
- lessons for other countries/labs
- source basis
