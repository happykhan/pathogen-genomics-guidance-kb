## Extracted/Adapted Text

The PHA4GE source lists data sharing as one part of data management and describes example implementation mechanisms such as local-only laptop workflows, HPC administrator-mediated sharing, cloud object storage, platform workspaces, Galaxy histories, and IRIDA projects.

Armstrong et al. support data-sharing content by describing public-health contributions to NCBI-hosted pathogen databases, ReSeqTB, and GISAID, while noting that data openness must be balanced against confidentiality and risk. The PHE case study gives a concrete example of Salmonella sequences being uploaded to NCBI SRA.

## Implementation Notes

Current source material supports platform examples, public database examples, and the need to balance openness with risk, but not a general repository or data-sharing decision framework.
