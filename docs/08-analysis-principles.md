# General Analytical Principles

## Core Principle

Analytical systems for pathogen genomics should be reproducible, interpretable, versioned, and fit for the public health decision they support.

## Minimum Expectations

- Distinguish exploratory analysis from routine surveillance analysis.
- Use documented pipeline versions for routine outputs.
- Track reference datasets and database versions.
- Record thresholds and decision rules.
- Preserve enough provenance to rerun or audit an analysis.
- Separate technical pipeline outputs from interpreted public health conclusions.
