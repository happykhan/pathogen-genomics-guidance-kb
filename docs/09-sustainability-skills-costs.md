# Sustainability, Skills, and Costs

## Core Principle

Sustainable pathogen genomics surveillance depends on people, operating processes, training, governance, and recurrent funding. Hardware or cloud credits alone are not a sustainable system.

## Planning Areas

- bioinformatics staffing
- data management
- platform administration
- workflow maintenance
- central IT support
- training and onboarding
- user support
- incident response
- procurement and renewal cycles
- storage and compute growth

## Common Failure Modes

- single-person dependency
- unbudgeted storage growth
- no maintenance ownership
- no validated update process
- platform selected without central IT buy-in
- tools deployed without user support
- infrastructure that works for pilots but not routine throughput
