# Data Standards and Metadata

## Core Principle

Genomics surveillance data become more useful when sample, laboratory, sequencing, analysis, and epidemiological metadata are structured, consistent, and linked to stable identifiers.

## Minimum Expectations

- Define required metadata fields for routine workflows.
- Capture metadata as early as possible.
- Use stable sample and analysis identifiers.
- Track relationships between sample, sequencing run, raw data, workflow run, results, and report.
- Use established standards where available.
- Keep local extensions documented.

## Open Work

Add disease-agnostic metadata recommendations and links to existing pathogen, AMR, and public health data standards.
