# Knowledge Base

This directory holds source-derived working material before it becomes assembled guidance.

## Folders

- `source-cards/`: provenance and source-pack indexes.
- `extracted-notes/`: cleaned source-backed notes.
- `figures/`: figure style rules, inventory, briefs, and later editable SVGs.
- `vignettes/`: implementation examples and source-backed vignette notes.
- `infrastructure/`: focused planning notes derived from source material.

## Status Rule

Every substantive note should state whether it is `source-backed`, `rough source`, `draft/gap`, or `skeleton`.

Do not add polished guidance here unless the source basis is explicit.
