# Source Card: International Pathogen Surveillance Network Source Pack

## Source basis

- Organization: International Pathogen Surveillance Network (IPSN), hosted by WHO.
- Main page: <https://www.who.int/initiatives/international-pathogen-surveillance-network>.
- Public beta PDF copy: `public/pdfs/ipsn-one-slide-overview.pdf`.
- Additional IPSN PDFs were reviewed in the local editorial workspace during source discovery.

## Status

Source-backed, but mostly programme/network framing rather than detailed technical guidance.

## Extracted Notes

- IPSN is framed as a global network bringing together pathogen genomic actors to improve public-health decision-making.
- The overview identifies communities of practice, country scale-up support, grant funding, advocacy/communications, and a global partners forum as IPSN functions.
- The catalytic grant calls are useful for understanding IPSN priorities, equity framing, capability building, and implementation support, but should not be treated as technical guidance.
- The public-consultation platform document has been superseded by the 2025 WHO platform attributes/principles report and should be used only as historical/provenance context.
