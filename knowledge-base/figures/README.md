# Figure System

## Style Rules

- Use black/grey linework with one restrained accent colour.
- Prefer simple tables, matrices, flows, and small multiples over dense posters or radar plots.
- Use short labels, consistent typography, and generous whitespace.
- Avoid gradients, decorative icons, 3D, stock graphics, and slideware styling.
- Keep editable SVG as the source format.
- Export PNG only for preview or downstream systems that cannot use SVG.
- Figures should explain one idea each.
